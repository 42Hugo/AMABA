Welcome to AMABA documentation!
===================================

**AMABA** provides an intuitive Graphical User Interface (GUI) for testing and experimenting with the additive manufacturing of aqueous phase materials.

Check out the :doc:`usage` to get started.

.. note::

   This project is under active development.
