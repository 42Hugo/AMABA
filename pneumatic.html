Pneumatic Control Documentation

Initialization

Pneumatic.__init__()[source]

The constructor method, comunication with EtherCAT is also started

Note

There is no control command for (c_point) as it maintains a fixed pressure between 4.7 and 7 bar to open the nozzle

Parameters:

  • st_cart (int) – set to 1 to open, 0 to close the solenoid valve on fluid, defaults to 0

  • st_ato (int) – set to 1 to open, 0 to close the solenoid valve on atomisation (to spay the liquid), defaults to 0

  • st_point (int) – set to 1 to open, 0 to close the solenoid valve on nozle (if closed atomization still works but no liquid is spayed), defaults to 0

  • c_cart (float) – pressure applied to the fluid between 0 and 2 bar (1 decimal possible), defaults to 0

  • c_ato (float) – pressure applied to the atomisation between 0 and 2 bar (1 decimal possible), defaults to 0

  • connected – indicates whether communication with the subprocess is established (self.connected = 2 when running), defaults to 0

  • filePath (str) – path to the gcode file, defaults to ‘’

  • host (str) – Hostname for the socket connection, defaults to ‘localhost’

  • port (int) – Port number for the socket connection, defaults to 12345

Send Methods

Pneumatic.sendToClient(state)[source]

Gathers all pressure-related information and sends it to the C process in the appropriate string format.

The state argument is always set to 1, except when terminating the process.

The string format is as follows:

  • The first bit indicates the state (1 or 0). A state of 0 will automatically terminate the loop.

  • Bits 2, 3, and 4 are for the solenoid valve: 0 for closed, 1 for open.

  • Bits 5, 6, 7, and 8 represent the pressure values (between 0 and 2) for atomization and fluid, without a comma.

The string should look like this:

mes = '[state][on/off atomization][on/off fluid][on/off nozzle][pressure atomization between 0 and 20][pressure fluid between 0 and 20]'

For example, consider the following string:

mes = '10001103'

  • The state is on (1).

  • All three solenoids are closed (0 for atomization, fluid, and nozzle).

  • 1.1 bar for atomization.

  • 0.3 bar for fluid.

Parameters:

state (int) – State of the process, 1 for running, 0 for stopping
